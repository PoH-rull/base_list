# Base Gear Inventory / ציוד בבסיס

A mobile-first Hebrew/English inventory tracker for managing base equipment.

## Features
- Track item quantities across categories
- Tag items to **Travel Bag** (🎒) or **Base Closet** (🗄️)
- Filter by status (full / missing), category, or location
- Live search
- "What to bring" checklist for missing items
- Export / import data as JSON
- Full Hebrew & English UI

## Usage
Open `index.html` directly in any browser — no install needed.  
Data is saved locally in your browser (localStorage).

**Live site:** https://poh-rull.github.io/base_list/
